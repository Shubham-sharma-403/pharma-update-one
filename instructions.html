03. Configure

As you might see in the /evident template directory, just we have one template; basic and expert:/ Basic starter: This medical filed is for simple websites which don't require any customization to TheDocs code and writing a few additional CSS and JS codes would be enough. Expert starter: The expert starter has several features which are not included in the basic starter. Those features give you more freedom in customizing your template and writing extra codes. It includes a necessary task to start a web server, watch files for changes and customize CSS and JS files. Finally, you can copy and paste to have all the required files for the template copied to a directory. We strongly recommend using the expert starter, but the basic starter template is advised for people who are not familiar with technical knowledge.
